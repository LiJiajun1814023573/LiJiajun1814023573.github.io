ES6前置知识(review系列)

· 2020-10-3 · 次阅读

ES6代码的转译

  1. 所需工具

    ES6语法编译成ES5以下语法:
babel-编译工具(理解为代码块，功能块)
需要npm安装

  2. npm安装babel相关工具

    babel-preset  我们指定的规则集为:babel-preset-env

  3. npm i babel-preset-env –save-dev安装到当前文件夹

  1. .babelrc文件(rc是run command运行命令的简写的)的配置
    {
 "presets": ["babel-preset-env"]
}

  1. 安装babel-cli –save-dev 安装babel脚手架工具(提供一系列命令)

    npm i babel-cli --save-dev

  2. 在”script”中添加

    "build": "babel app.js -o bundle.js"　

    指示babel转义后output到bundle.js

      "babel src -d lib"

  3. 命令行中使用命令,进行转义

npm run build

app.js

var arr = [];

for(let i = 0; i < 10; i++){
    arr[i] = (() => {
        console.log(i);
    })(i)
}

配置执行相应的文件

"script-name": "babel-node ./src/app.js"

或者执行也可以直接用node

babel更多工具

babel-core：让部分代码转码

babel-register：利用Require钩子自动编译

app.js

var babel = require("babel-core");

app.js


require("babel-register");

var obj = new Map(); // 新的API没有被转码，通过babel-polyfill插件进行转换成相应浏览器能的代码

console.log(obj);

babel-polyfill:

require("babel-polyfill");

其他方式转义

也可以通过babel提供的在线的工具，获取浏览器兼容的babel后的js代码

浏览器当中使用ES6代码
在浏览器中引入cdn文件,babel官网提供

<script type="text/babel">
// 代码
</script>

Traceur
是谷歌提供的转码工具。
在浏览器中使用，同样需要引入script文件

https://node.green
node关于ES6语法支持情况不错
因此我们使用node环境测试ES6代码即可
